# Руководство пользователя

Версия Node.js : v14.18.1

## Устанвка приложения

Выполните команды в терминале:
### `git clone -b main https://github.com/domino1198/testProject`
### `npm i` 
or 
### `yarn`


## Запуск приложения

Выполните команду:
### `npm dev`
or
### `yarn dev`
